## Cartella vendor
Per generare la cartella vendor devi:
- scaricare il programma composer da https://getcomposer.org/download/
- eseguire il comando composer install dalla radice del progetto
- caricare manualmente via ftp la cartella vendor alla radice del server web
- caricare manualmente via ftp i file composer.lock e composer.json alla radice del server web
